## AWS Elemental Live

Control the AWS Elemental Live encoder.

## Configuration

- Enter the encoder's IP address in the module settings

_Note: At this time, encoders with authentication enabled are currently not supported_

## Actions

- Start Live Event
- Stop Live Event
- Cancel Live Event
- Archive Live Event
- Reset Live Event
- Mute/Unmute Audio
- Set Audio Gain

## Feedback

- Live Event Status
- System Status

## Variables

- system_cpu
- system_memory
- system_gpu
- event\__id_\_name
- event\__id_\_status
- event\__id_\_duration
